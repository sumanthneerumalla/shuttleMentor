# Screen Inventory for Badminton Coaching Platform MVP

## General/Shared Screens
1. **Landing Page**
   - Value proposition
   - Call-to-action buttons (Sign up as Coach/Student)
   - Featured coaches
   - How it works section
   - Students section with faq
   - Coaches section with faq

2. **Sign Up/Login**
   - Email/password fields (using clerk api)
   - Role selection (Coach/Student)
   - Credit card details ()
   - Terms & conditions

3. **Dashboard (Student/Coach/Admin views)**
   - Overview of upcoming sessions
   - Notifications
   - Quick actions

## Student Screens
4. **Student Profile Setup**
   - Personal information
   - Experience level
   - Goals
   - Time Zone
   - Profile photo
   - Payment details and management

5. **Coach Browse & Search**
   - List of coaches with filtering options
   - Coach cards with key information

6. **Chat management**
   - List of chat conversations
   - Search bar to select chat recipients
   - Hover on name to view details of chat contacts
   - Book session button

7. **Coach Profile View**
   - Coach details, experience, specialties, images or video uploads
   - Hourly rate
   - Availability calendar
   - Book session button

8. **Booking Screen**
    - Date/time selection
    - Coach selection
    - Session duration
    - Confirmation

9. **Video Upload**
   - Upload interface
   - Grouping functionality
   - Description fields, naming videos

10.  **Past Sessions**
    - List of completed sessions
    - Access to recordings
    - Coach notes
    - AI summaries
